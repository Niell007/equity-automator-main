### Add

- Core dashboard layout implementation
  - Responsive sidebar navigation
  - Top navigation bar
  - Theme switching support
- Main dashboard overview page
  - Statistics cards
  - Quick actions
  - Recent activity feed
  - Upcoming deadlines
- Workforce management page
  - Employee data table
  - Filtering system
  - Import/Export functionality
- Reports generation page
  - Report type cards
  - Configuration options
  - Report history
- Analytics dashboard
  - Overview statistics
  - Workforce distribution charts
  - Department analysis
  - Key insights panel
- Settings page
  - Profile management
  - Notification preferences
  - Security settings
  - Organization configuration
- UI Components
  - StatCard component
  - DataTable component
  - Form elements
  - Layout components
- Theme system
  - Light/dark mode support
  - CSS variable-based theming
  - Consistent color palette

- Theme switching edge cases
